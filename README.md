# 📝 TaskMaster: Minimalist To-Do List

Selamat datang di **TaskMaster**! Sebuah aplikasi pencatatan tugas sederhana namun elegan yang dibangun khusus untuk fokus pada sisi *frontend*. Proyek ini dirancang untuk memaksimalkan produktivitas dengan antarmuka yang bersih dan responsif.



## 🚀 Fitur Utama
*   ✨ **Desain Modern:** Menggunakan Tailwind CSS untuk tampilan yang *clean*.
*   📱 **Fully Responsive:** Nyaman digunakan di HP, Tablet, maupun Desktop.
*   ⚡ **Interaktif:** Tambah, coret, dan hapus tugas secara instan.
*   🎨 **Custom Themes:** Skema warna yang memanjakan mata.

---

## 🛠️ Stack Teknologi
Proyek ini murni menggunakan teknologi *frontend* untuk kecepatan dan efisiensi:

| Komponen | Teknologi |

## 📋 Alur Kerja (Workflow)

Berikut adalah tahapan pengembangan yang saya gunakan dalam membangun proyek ini:

### 1. Planning & Wireframing 🎨
Menentukan struktur dasar aplikasi (Input field, tombol tambah, dan list kontainer).

### 2. Environment Setup ⚙️
Menginisialisasi Tailwind CSS melalui CDN atau CLI.
```bash
npm install -D tailwindcss
npx tailwindcss init
```

### 3. Slicing UI ✂️
Menyusun elemen HTML dan memberikan *utility classes* dari Tailwind untuk styling layout seperti `flex`, `grid`, `padding`, dan `margin`.

### 4. Logic Implementation 🧠
Menulis script JavaScript untuk:
*   Menangkap input pengguna.
*   Menambah elemen `<li>` baru ke dalam DOM.
*   Menghapus tugas menggunakan *event listener*.

### 5. Deployment 🌐
Mengunggah hasil akhir ke platform hosting statis seperti **GitHub Pages**, **Vercel**, atau **Netlify**.
